基于 hexo-github 的blog搭建方案

之前的 blog 一直使用的是 wordpress，搭建使用起来还算方便。但是wordpress 漏动较多、需要备份 mysql 等等，维护起来比较麻烦，所以准备迁移到比较轻量的 hexo。

hexo是一套基于 nodejs 的简单轻量的blog框架，使用它，可以直接本地编辑好 markdown 文件，然后上传到 github 或者 gitlab 上，hexo 直接从这些源拉取文件，省去了维护存储的成本。

本文总结了使用 hexo-github-nginx 搭建个人blog的方法。

Prepare

在安装 hexo 之前，需要做一些前期的准备工作，安装一些必备的依赖和组件

（系统 Centos 7）

  1. 安装 nodejs

  2. 安装 nginx

  3. 安装 git

  4. 创建拥有 sudo 权限的用户

  5. 申请一个有效的 github 账户
